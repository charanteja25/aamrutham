# Aamrutham React App

A cleaner multi-page React version of the Aamrutham site.

## Included pages

- Home page
- Products listing page
- Product detail page
- Shared cart drawer with localStorage
- WhatsApp pre-order flow

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- No Tailwind or PostCSS is required.
- The app uses plain React + React Router + CSS.
- Brand artwork is included from `public/assets/Subject.png` and `public/assets/aam-final.png`.
