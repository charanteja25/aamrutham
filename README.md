# Aamrutham

A Natural, Organic Fruit selling application.

## Run locally

* setup Razorpay keys in .env file by copying from .env.example

```bash
npm install
npm run dev
```

backend
```bash
cd backend
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- No Tailwind or PostCSS is required.
- The app uses plain React + React Router + CSS.
- Brand artwork is included from `public/assets/Subject.png` and `public/assets/aam-final.png`.
